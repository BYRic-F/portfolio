# Portfolio Data Analyst - Frédéric Bayen

Ce projet est un portfolio interactif réalisé avec **Streamlit**. il présente mes travaux en Data Analysis, Machine Learning et Business Intelligence.

## Structure du projet

- `main.py` : Point d'entrée de l'application Streamlit.
- `style.css` : Styles personnalisés (centrage des titres, cartes, etc.).
- `.streamlit/config.toml` : Configuration du thème (couleurs, polices).
- `data/` : Dossier contenant les données (READMEs, images, présentations PDF, CV).

## Installation

Ce projet utilise `uv` pour la gestion des dépendances.

1. Installez les dépendances :
   ```bash
   uv sync
   ```

2. Lancez l'application :
   ```bash
   uv run streamlit run main.py
   ```

## Contenu

- **Page d'accueil** : Présentation et compétences.
- **Projets Python** : Détection de fraude, IA Prospector, Système de recommandation.
- **Projets Power BI** : Optima Cycles, Video Game Market, Accès à l'Eau, Toys & Models.
- **CV** : Visualisation et téléchargement de mon CV.
